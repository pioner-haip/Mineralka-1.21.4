Описание (Description)Этот мод революционизирует взаимодействие с метаданными блоков путем инвертирования квантового состояния эндер-сундука. Используя наш запатентованный алгоритм void-pointer-reference-fixer, мод переписывает рендер-движок прямо во время генерации чанка.Основные фичи:[ВЕЩЬ] Интеллигентный блок грязи: он смотрит на вас, когда вы на него не смотрите.[МЕХАНИКА] Автоматическая конвертация алмазов в виртуальные очки опыта, которые нельзя потратить.[BLOCK] Ступеньки, которые ставятся только в четвертом измерении.[API] Совместимость с: Forge, Fabric, Quilt, Liteloader, Optifine, Sodium, а также с модом на атомную бомбу, установленным у соседа.
